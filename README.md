# Little Rock Crime Mini Project

This repository contains Mini Project #1 for GIS Programming.

## Project overview
This project analyzes the spatial and temporal distribution of reported crime incidents in Little Rock, Arkansas, from 2020 to 2025. The workflow includes data cleaning, null-value screening, removal of incomplete 2026 records, spatial filtering, exploratory plots, static mapping, and ArcGIS Online publishing using the ArcGIS Python API.
